# NextJS Sample App
This is a simple sample app built with NextJS and Postgres.

## Setup with Docker (recommended)
```bash
docker-compose up
```

## Setup without Docker
1. [Install Postgres](https://www.postgresql.org/download/macosx/)
2. Install Node 16 
3. Install Yarn
4. Install dependencies
```bash
yarn
```

## Running the app
```bash
yarn dev
```
